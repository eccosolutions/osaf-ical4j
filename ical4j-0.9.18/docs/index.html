Overview »

n

n iCal4j is a Java library used to read andn write iCalendar data streams as defined inn RFC2445.n The iCalendar standard provides a common data format used to store information aboutn calendar-specific data such as events, appointments, to-do lists, etc.n All of the popular calendaring tools, such as Lotus Notes, Outlook and Apple's iCaln also support the iCalendar standard.n

n

n Providing both a parser and an object model, iCal4j allowsn you to either modify existing iCalendar data or create newn data models. Validation is also provided to ensure then data maintains a state consistent with the specification.n
